# Icebreaker Project

This project aims to gather live user details from Wikipedia through a scraping process using **LangChain** and the **Gemini model** to provide accurate and relevant information. The project is designed to remove the need for extensive reading by automating the process of retrieving and presenting key details about users. 

## Features

- Scrapes Wikipedia for user details based on input criteria.
- Uses LangChain for efficient data retrieval and parsing.
- Incorporates the Gemini model for extracting accurate, high-quality information.
- Eliminates manual reading by automating the extraction of essential details.
- Written in Python for ease of integration and modification.
